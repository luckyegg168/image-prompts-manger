# PromptMaster Pro — 詞譜專家

AI 繪圖提示詞管理器，使用 Python + NiceGUI 打造。

## 功能

- 🎨 **模組化標籤詞庫** — 8 大類別、140+ 內建標籤
- 🔌 **LoRA 觸發詞管理** — 34+ 預設 LoRA (動漫/寫實/特殊/角色IP/FLUX)
- ⚖️ **精確權重控制** — 每個標籤可獨立調整 0.1~2.0
- 💾 **預設組管理** — 儲存/載入/刪除常用提示詞組合 (含 5 組內建範例)
- 🎲 **隨機擾動器** — 鎖定核心標籤，隨機替換其餘
- 🌙 **深色模式** — 預設深色，保護眼睛
- 📤 **匯出/匯入** — JSON 格式備份與還原

## 快速開始

```bash
pip install -r requirements.txt
python main.py
```

瀏覽器自動開啟 → `http://localhost:8080`

## 技術棧

- Python 3.10+
- NiceGUI 3.9+ (Quasar + Vue + Tailwind)
- JSON 文件持久化

## 截圖

啟動後即可看到三欄式介面：
- **左側**：類別導航 + LoRA 管理
- **中間**：標籤詞庫（點擊即選取）
- **右側**：提示詞工作台（即時預覽 + 權重調整）
